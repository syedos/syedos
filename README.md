# syedOS

Personal site for syedOS — an interactive desktop-style portfolio with a boot sequence, draggable windows, terminal, music player, and GitHub activity widget.

## Tech Stack

- React + TypeScript + Vite
- Tailwind CSS + shadcn/ui
- Framer Motion

## Development

```sh
git clone https://github.com/syedos/syedos.git
cd syedos
npm i
npm run dev
```

Runs on `http://localhost:8080`.

## Build

```sh
npm run build
```
